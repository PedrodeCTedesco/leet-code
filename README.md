# Algoritmos

## Diferenças entre algoritmos
### Diferença de Two Pointers e Prefix/Suffix

| Aspecto | Two Pointers | Prefix/Suffix |
|---------|--------------|---------------|
| **Foco** | Manipular **posições** | Acumular **valores** |
| **Movimento** | Decisões dinâmicas | Direções fixas (esq→dir, dir→esq) |
| **Invariante** | Relação entre ponteiros | Agregação progressiva |
| **Uso típico** | Busca, comparação, partição | Somas, produtos, queries |
